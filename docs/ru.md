# Sweet Spot - Полное руководство пользователя

Добро пожаловать на платформу клиентской аналитики Sweet Spot! Это руководство объяснит, как использовать все функции на сайте для извлечения максимальной пользы из данных о ваших клиентах.

## 1. Панель управления (Dashboard)
Панель управления дает вам полный обзор вашей клиентской базы.
- **Сетка метрик**: Показывает общее количество клиентов, средний балл, % лучших клиентов и общий доход.
- **График "Прибыльность vs Усилия"**: Квадрантный график, отображающий ваших клиентов в зависимости от того, сколько усилий они требуют, по сравнению с тем, какую прибыль приносят. Верхний левый квадрант представляет "Зону оптимального роста" (высокая прибыль, низкие усилия). "Лучшие" клиенты будут автоматически пульсировать на графике.
- **Разбивка по отраслям**: Отображает распределение ваших клиентов по различным отраслям на линейчатой диаграмме.
- **Топ-10 клиентов**: Список ваших клиентов с наивысшими баллами на основе весов критериев, которые вы установили в Настройках.

## 2. Клиенты (Сырые данные)
Здесь вы управляете вашим списком клиентов.
- **Добавить клиента**: Нажмите эту кнопку, чтобы добавить нового клиента. Вы укажете основную информацию (Имя, Отрасль) и объективные метрики (Доход, Маржа прибыли в евро, Усилия в часах).
- **Субъективная оценка**: При добавлении или редактировании клиента вы также оцениваете его по субъективным чертам:
  - Оценка совместимости/химии (1-5)
  - Потенциал роста (1-5)
  - Готовность платить (1-5)
  - Процент повторных сделок (%)
  - Количество рекомендаций
Эти метрики комбинируются нашим алгоритмом для расчета общего балла "Sweet Spot" для каждого клиента.

## 3. Настройки (Весовые коэффициенты оценок)
Это ядро алгоритма Sweet Spot.
- Вы можете назначать определенные процентные веса (от 0% до 100%) для 7 различных критериев: Прибыльность, Усилия, Совместимость, Потенциал роста, Готовность платить, % Повторных сделок и Рекомендации.
- **Важно**: Общий вес всегда должен быть равен ровно 100%.
- Как только вы нажмете "Сохранить веса", система пересчитает баллы и рейтинг для каждого клиента на основе ваших новых предпочтений. Это напрямую обновит список "Топ-10" на Панели управления.

## 4. Команда и роли
Управляйте тем, кто имеет доступ к вашей рабочей области.
- **Пригласить пользователей**: Отправьте приглашения по электронной почте своим коллегам для присоединения.
- **Роли**: Назначьте роль "Администратор" или "Пользователь". Администраторы могут управлять другими пользователями, удалять записи и изменять конфигурацию системы, тогда как обычные пользователи имеют ограниченный доступ. ОРИГИНАЛЬНЫЙ создатель аккаунта защищен навсегда как Администратор.

## 5. Многоязычная поддержка
Sweet Spot доступен на 10 языках. Вы можете изменить свои языковые предпочтения в верхней панели навигации или в подвале сайта для мгновенного перевода всех элементов.
